# IT-Lab-Portfolio
IT Lab Portfolio — Hands-On Troubleshooting

This repository documents hands-on IT troubleshooting labs completed while preparing for CompTIA A+.

The labs simulate real Help Desk scenarios including:

Network connectivity failures

DNS troubleshooting

Authentication and account lockout investigations

Service and dependency failures

Device/driver troubleshooting

Performance and storage investigations

Each lab includes scenario, investigation steps, root cause, and resolution.
